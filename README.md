# 🌐 Network Scanner (Nmap Automation)

🚀 A Python-based tool to automate network reconnaissance using Nmap.

## 🔍 Features
- Scan single IP and subnet (e.g., 192.168.1.0/24)
- Detect open ports and services
- OS detection
- Export results in JSON format

## 🧠 Skills Demonstrated
- Network Enumeration
- Reconnaissance
- Python Automation
- Cybersecurity Fundamentals
