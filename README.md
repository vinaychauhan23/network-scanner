# 🌐 Network Scanner (Nmap Automation)

## 📌 Description
This project is a Python-based network scanner that automates Nmap to detect open ports, running services, and operating system details of a target system.

## 🚀 Features
- Scan single IP or network
- Detect open ports
- Identify services (HTTP, SSH, FTP, etc.)
- OS detection
- Save scan results to file

## 🛠️ Technologies Used
- Python
- Nmap
- python-nmap

## ▶️ Usage

### Install dependencies:
pip install -r requirements.txt

### Run the scanner:
python scanner.py

or

python scanner.py 127.0.0.1
